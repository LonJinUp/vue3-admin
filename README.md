# vue3-admin

> 基于vue3的后台管理系统

## 插件

[vue3](https://cn.vuejs.org/)  
[pinia](https://pinia.vuejs.org/zh/)  
[vue-router](https://router.vuejs.org/zh/)  
[pinia-plugin-persistedstate](pinia-plugin-persistedstate) [lodash](https://www.lodashjs.com/)

## 启动项目

初始化项目

```sh
npm install
```

运行

```sh
npm run dev
```

## 打包

```sh
npm run build
```
